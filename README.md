Kikurage Cultivation Device  
===

## Development

### Setup

- Install PlatformIO by VSCOde plugin
- Install M5Stack libraries
  - PIO Home > Libraries > M5Stack > Add Project
  - After that theses are set in /.pio/libdeps

### Run

1. Build
2. Write to devices
